**RevHire** is a **console-based job portal application** that connects **job seekers** with **employers**.  
It enables job seekers to create profiles, build resumes, search and apply for jobs, while employers can post job openings, manage applications, and shortlist or reject candidates.

The application follows a **modular architecture**, making it scalable and extendable to a **microservices-based web application** in future phases.
